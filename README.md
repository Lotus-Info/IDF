## 🧠 About the App

Issue Duplicate Finder for Jira helps teams improve productivity by surfacing similar Jira issues in real time directly inside the issue view.

It enables teams to:
- Reduce duplicate work  
- Reuse existing solutions  
- Accelerate issue resolution  
- Improve visibility across projects  

The app is built on **Atlassian Forge** and runs securely within the Atlassian ecosystem.

## 🐞 Reporting Bugs

If you encounter a bug or unexpected behavior, please create an issue here:

https://github.com/Lotus-Info/IDF/issues

Please include the following information:

- Jira Cloud instance (if applicable)  
- Steps to reproduce the issue  
- Expected behavior  
- Actual behavior  
- Screenshots or recordings (if possible)  
- Any error messages or logs
